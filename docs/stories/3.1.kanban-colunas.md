## Story
**As a** Anderson (gestor da 4G),
**I want** ver todos os leads organizados no Kanban por status,
**so that** eu possa acompanhar o funil de qualificação em tempo real sem abrir planilha nenhuma.

## Acceptance Criteria
1. Kanban exibe 4 colunas: `PROSPECTADOS`, `INTERESSE`, `TRANSFERIDOS`, `DESCARTADOS`
2. Cada card exibe: empresa, telefone, cidade, categoria (DOMÉSTICOS/ESPORTIVOS/MISTO), nota da IA (0-10) e data de coleta
3. Contador de leads visível no cabeçalho de cada coluna
4. Leads carregados do Supabase e atualizados em tempo real via Realtime
5. Layout responsivo — desktop (4 colunas lado a lado) e celular (colunas empilhadas)

### Supabase Realtime — Kanban
[Source: docs/architecture.md#tech-stack]
```typescript
// Subscription para todas as mudanças em leads
supabase
  .channel('kanban-leads')
  .on('postgres_changes', {
    event: '*',  // INSERT, UPDATE, DELETE
    schema: 'public',
    table: 'leads'
  }, (payload) => {
    // Atualizar mapa de leads por status
  })
  .subscribe()
```
