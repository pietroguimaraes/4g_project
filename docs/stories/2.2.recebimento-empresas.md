## Story
**As a** sistema,
**I want** receber as empresas encontradas pelo Apify e exibi-las como cards com status `LOCALIZADOS`,
**so that** o Anderson possa revisá-las antes de qualquer WhatsApp ser disparado.

## Acceptance Criteria
1. Endpoint da API recebe as empresas enviadas pelo n8n após a busca (autenticado por API key)
2. Empresas salvas na tabela `leads` com status `LOCALIZADOS`
3. Cards exibidos no Painel de Aprovação com: nome da empresa, telefone e cidade
4. Painel atualiza automaticamente quando novas empresas chegam (Supabase Realtime)
5. Contador de empresas com status `LOCALIZADOS` visível no painel

### Segurança da API Key

```typescript
// Validação no endpoint /api/leads
const apiKey = request.headers.get('x-api-key')
if (apiKey !== process.env.N8N_API_KEY) {
  return Response.json({ error: 'Unauthorized' }, { status: 401 })
}
```

### Supabase Realtime
[Source: docs/architecture.md#tech-stack]
```typescript
// Subscription para novos leads LOCALIZADOS
supabase
  .channel('leads-localizados')
  .on('postgres_changes', {
    event: 'INSERT',
    schema: 'public',
    table: 'leads',
    filter: "status=eq.LOCALIZADOS"
  }, (payload) => {
    // Adicionar novo lead à lista
  })
  .subscribe()
```
