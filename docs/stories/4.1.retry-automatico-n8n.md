## Story
**As a** sistema,
**I want** executar rodadas adicionais de busca no Apify automaticamente quando o número de leads válidos for menor que o pedido,
**so that** o Anderson receba o máximo possível de leads sem precisar reconfigurar a busca.

## Acceptance Criteria
1. Após filtrar leads da primeira rodada, se `leads_validos < quantidade_pedida`, o fluxo executa nova rodada com `maxResults` maior
2. O sistema executa até 3 rodadas antes de encerrar
3. Dedup por telefone funciona entre todas as rodadas (sem duplicatas)
4. O fluxo para imediatamente quando `leads_validos >= quantidade_pedida`
5. Ao encerrar, o fluxo chama PATCH `/api/searches/:id` com `status`, `quantidade_bruta`, `quantidade_entregue`, `num_rodadas`
