## Story
**As a** Anderson (gestor da 4G),
**I want** ver a estrutura base do dashboard após o login,
**so that** eu possa confirmar que o sistema está funcionando e navegar entre as seções.

## Acceptance Criteria
1. Navbar exibida no topo com nome do projeto ("4G Dashboard") e botão de logout
2. Menu de navegação com dois itens: "Painel de Aprovação" (`/aprovacao`) e "Kanban" (`/kanban`)
3. Página `/aprovacao` renderiza sem erros (tela vazia, sem dados)
4. Página `/kanban` renderiza sem erros (tela vazia, sem dados)
5. Layout responsivo — funciona em desktop e celular (mobile-first)

## Dev Notes

### Estrutura de Arquivos
[Source: docs/architecture.md#estrutura-do-projeto]
```
src/app/
└── (dashboard)/
    ├── layout.tsx          # Layout compartilhado — Navbar + nav
    ├── aprovacao/
    │   └── page.tsx        # Painel de Aprovação (placeholder)
    └── kanban/
        └── page.tsx        # Kanban de Funil (placeholder)
```

### Componentes a Usar
[Source: docs/architecture.md#tech-stack]
- **shadcn/ui:** `Button`, `Link` — componentes de UI base
- **Next.js `<Link>`:** navegação entre páginas sem reload
- **Tailwind CSS:** estilização responsiva com classes mobile-first
- **`<LogoutButton />`:** componente criado na Story 1.3 (`src/components/ui/LogoutButton.tsx`)

### Layout do Dashboard
[Source: docs/architecture.md#componentes]
```typescript
// src/app/(dashboard)/layout.tsx
import { LogoutButton } from '@/components/ui/LogoutButton'
import Link from 'next/link'

export default function DashboardLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <div className="min-h-screen bg-gray-50">
      <nav className="bg-white border-b px-4 py-3 flex items-center justify-between">
        <div className="flex items-center gap-6">
          <span className="font-bold text-lg">4G Dashboard</span>
          <Link href="/aprovacao">Painel de Aprovação</Link>
          <Link href="/kanban">Kanban</Link>
        </div>
        <LogoutButton />
      </nav>
      <main className="p-4">{children}</main>
    </div>
  )
}
```
